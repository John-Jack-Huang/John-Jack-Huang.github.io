嘿，你好~
我们又相遇了，正如我们又要匆匆一别。
致路过的每一个你，所有的不期而遇都是久别重逢，人生的遇见是一种缘分。你能看此文，无非两种可能：

  1. 你是我最信任的朋友，所以我告诉你访问网址；
  2. 你是千百度中一人，百度将你引领到这里。

无论你来自何方，接下来无论去往哪里，正如标题里：你好呀，朋友！

关于木屋

为什么这个网站叫做木屋呢？木屋是一个房子，是我家一个工作的地方，小时候要帮父母干活，经常呆在那。可以说在这个屋子度过了我整个童年，直到初中的时候，搬了家，学业也逐渐繁重，我就很少进去这个工作坊帮父母干活了。那里有过一家人完整的团聚，有过我的汗水与对其环境的嫌弃，有过我和姐姐的打闹。如今家里还依旧有一个工作坊，家人还时常在里面工作，而我每年回家的时候，那个工作坊大多变成了烤火的地方，所以命此木屋。源于童年，关于青春，将于未来。

关于自己

  1. 文学伪爱好者，Github非专业搬运工；
  2. 没读过多少书籍，容易陷入回忆死循环；
  3. 一杯热水一支笔，一个Bug改一天；
  4. 极端拖稿，老师不催，我的思路就无比堵塞；
  5. 熬夜猝死，深夜的酒真好喝！深夜睡不着么，来听电台吧；
  6. em….还有挺多的缺点！

啥时候改了，我再回来添上。

关于《手中流沙，纸上年华》

《手中流沙，纸上年华》其实是我高中的一本读书笔记。高中时期的所有作文文稿大多都在里面，这本读书笔记，见证了我三年的青春，还在大学里陪伴了我四年，虽然说大学里不再在这本笔记上写东西了，但是它依旧悄悄立在我头顶的书柜里。这网站的首页就是手中流沙，点击开始阅读，就会跳到纸上年华。希望这个网站能像高中时期的笔记本一样，也能记住我的青春时刻。

关于本站

其实很久之前就想拥有一个网站了，来记录一些学习的笔记和自己的日常。但是以前觉得各种配置特别困难，也不知道该如何弄。然后前段时间在简书看到一个从零搭建个人博客的帖子，可以说这个就是我入门搭建博客的文章了，虽说现在也完全不懂前端的知识，但是自己改改网站里的主题、颜色，也是没问题了，这对我来说已经足够了。
后来根据黄玄大佬的教程，弄了一个网站，部署在Github上面，当时第一次看到网页上显示，真的挺激动的。部署在Github的原博客链接。
原木屋


后来用了几天之后，对这个网站的功能不是太满意。
第一是没有返回顶端的功能，因为文章一长，返回顶部很麻烦。后来自己折腾加了个返回顶部的功能，点击图中右下角的那个小旺仔，就可以返回顶部。但是这只能在手机端显示，而PC端，会被拦截，无法显示。我猜可能是被识别成广告弹出，然后被浏览器拦截了。

第二是前些天，在Hexo主题里浏览的时候，看到了这款Matery主题，最重要的是，博主写了超级详细的教程，这对于我这种没有一点基础的人来说（其实好像也有，前期安装了Nodejs 和Hexo，但是没用过），简直是超级大的好事啊！！

其实还有很多，期间出的错误，大多都是网上查的教程解决的，感谢两位大佬黄玄，闪烁之狐，感谢那些在网上刊出自己错误经历、乐于分享的人，让我学习到了很多。多去Github给大佬们标星星，让优秀的作品获得更多人的注意。

最后呢，我没有像原来一样，把网站部署到Github上，而是部署到国内的码云上。因为呀，Github服务器在国外，我的天！访问实在是太慢了！然后换到部署在码云上，速度相比之下真的快的飞起，很流畅！！

关于想说

  1. 用程序员最基本的方式说声：

    void John_Jack_Said(){
    System.out.print("Hello World!");
}

    无论学习到什么新的知识，第一句都要是：Hello World。致敬世界，对世界保持一颗敬畏之心。

  2. 用文艺青年的方式说声：人生若只如初见，何事秋风悲画扇。

关于最后

谢谢你，能看到这里。那么，谨祝我们：
  • 永远年轻，永远热泪盈眶！
