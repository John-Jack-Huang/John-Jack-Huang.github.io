1.反向传播

反向传播是用于训练模型参数，在所有参数上使用梯度下降法，使得神经网络模型在训练数据上的损失函数值最小。下面介绍一下反向传播需要到的损失函数、训练方法等。

2.损失函数（Loss）

损失函数是表示预测值（y）和**标准值（y_ )**之间的差距值。
损失函数有很多种计算方法，下面的是比较常用的方法。

2.1用均方误差（MSE）表示损失函数

均方误差（MSE）是求前向传播的计算结果$y$与标准值$y
_-$之间差的平方和，再求平均。
均方误差（MSE）计算公式：
$$ MSE(y,y_-) = \frac{\sum_{i=0}^n (y - y_-)^2}{n} $$
TensorFlow函数表示为：

loss_mse = tf.reduce_mean(tf.square( y - y_ ))

2.2用交叉熵(Cross Entropy)表示损失函数

交叉熵表示两个概率分布之间的距离。交叉熵越大，两个概率分布距离越远，两个概率分布越相异；交叉熵越小，两个概率分布距离越近，两个概率分布越相似。
交叉熵（ce）计算公式：$$H(y,y_-)=- \sum{y_- \times \log y } $$
TensorFlow函数表示为：

loss_ce = -tf.reduce_mean(y_* tf.log(tf.clip_by_value(y, 1e-12, 1.0)))

其中，为了保证$\log$有意义，tf.clip_by_value(y, 1e-12, 1.0)表示当$y$小于$10^{-12}$时，$y$为$10^{-12}$；当$y$大于1时,则为1。
因为我们的输出一般都不是概率分布，所以我们要先把输出的值转化为概率分布，才能用交叉熵（ce）来表示损失函数。在TensorFlow中，一般让模型的输出经过softmax函数，用于获得输出分类的概率分布。再与标准值$y_-$对比，求出交叉熵，得到损失函数。下面介绍一下softmax函数。
softmax函数：将n分类的n个输出（$ y_1,y_2,y_3…..,y_n$；$y_i$表示第$i$中情况出现的可能性大小)变为满足以下概率分布的函数：$$ \forall x , P(X =x)\in[0,1]且\sum{P_x{(X=x)}}=1 $$
softmax函数数学表示：$$ softmax(y_i)=\frac{e^{y_i}}{\sum_{j=1}^n {e^{y_i}}}$$
让模型输出经过$softmax$函数，再与标准值$y_-$做交叉熵，得到损失函数。TensorFlow函数表示为：

#输出值经过softmax函数，再与标准值得到的交叉熵ce
ce = tf.nn.sparse_softmax_cross_entropy_with_logits(logits=y, labels=tf.argmax(y_, 1))
cem = tf.reduce_mean(ce)#损失函数值

3.反向传播训练方法

我们是以减少$loss$值为训练的优化目标,优化方法主要有以下几种：

3.1梯度下降方法

使用随机梯度下降算法，使参数沿着梯度的反方向，即总损失减小的方向移动，实现更新参数。
参数更新公式：$$ \emptyset_{n+1} = \emptyset_{n}- a\frac{\partial J(\emptyset_{n})}{\partial \emptyset_{n}} $$
其中，$ J(\emptyset)$为损失函数，$ \emptyset$为参数，$a$为学习率。
代码表示:

'''梯度下降方法'''
#learning_rate为学习率，决定参数每次更新的幅度
#loss为损失函数值，即上面提到的loss_mse = tf.reduce_mean(tf.square( y - y_))
train_step=tf.train.GradientDescentOptimizer(learning_rate).minimize(loss)

3.2momentum 优化器

momentum 优化器在更新参数时，利用了超参数，参数更新公式是:$$ d_i =\beta d_{i-1} +g(\emptyset_{i-1}) $$
$$ \emptyset_{i} = \emptyset_{i-1} -ad_i$$
其中，$a$为学习率，$\beta $为超参数，$\emptyset$为参数，$g(\emptyset_{i-1})$为损失函数的梯度。
代码表示:

'''momentum 优化器优化方法'''
#learning_rate为学习率，决定参数每次更新的幅度
#loss为损失函数值
train_step=tf.train.MomentumOptimizer(learning_rate,momentum).minimize(loss)

3.3adam 优化器

adam 优化器是利用自适应学习率的优化算法，Adam算法和随机梯度下降算法不同。随机梯度下降算法保持单一的学习率更新所有的参数，学习率在训练过程中并不会改变。而Adam算法通过计算梯度的一阶矩估计和二阶矩估计而为不同的参数设计独立的自适应性学习率。
代码表示：

'''adam 优化器优化方法'''
#learning_rate为学习率，决定参数每次更新的幅度
#loss为损失函数值
train_step=tf.train.AdamOptimizer(learning_rate).minimize(loss)

4.学习率

定义：学习率是决定每次参数更新的幅度。
由以上提到的3种优化方法得知，每一个方法都需要一个叫做学习率的参数，在选择学习率的时候，如果学习率过大，则会出现震荡不收敛的情况；如果学习率过小，会出现收敛速度过慢的情况，不利于快速得到结果。所以一般选择学习率为：0.01、0.001.

4.1固定学习率

所谓固定学习率就是一个定值，不会动态更新。每次迭代的过程中，学习率保持不变。
参数更新方式：$$ w_{n+1} =w_n -learning_-rate \times \nabla $$
假设损失函数为：$loss = （w+1）^2$, 上式中的 $\nabla$是损失函数的导数：$2(w+1)$。假设初始值$w$=2，学习率$learning_-rate$=0.2,那么经过3轮迭代之后参数$w$如下：

迭代次数 更新前参数w 计算过程 更新后的参数w
0次 2 0.8=2-0.2x2x（2+1） 0.8
1次 0.8 0.08=0.8-0.2x2x（0.8+1） 0.08
2次 0.08 -0.35=0.08-0.2x2x（0.08+1） -0.35

用代码表示：

'''固定学习率'''
import tensorflow as tf
learning_rate =0.2#学习率
#定义参数，初始化为2
w = tf.Variable(tf.constant(2,dtype=tf.float32 ))
#定义损失函数loss = （w+1）^2
loss = tf.square( w + 1)
#定义反向传播方法
train_step=tf.train.GradientDescentOptimizer(learning_rate).minimize(loss)

with tf.Session() as sess:
    #初始化所有变量
    init_op = tf.global_variables_initializer()
    sess.run(init_op)
    
    STEPS = 20#训练20轮
    for i in range(STEPS):
        sess.run(train_step)
        w_value = sess.run(w)
        loss_value = sess.run(loss)
        print("迭代{}次，w的值为：{}f，损失值为：{}f".format(i,w_value,loss_value))

4.2指数衰减学习率

指数衰减学习率是指学习率随着训练轮数动态更新。
学习率计算公式如下：
$$learning_-rate = learning_-rate_-base \times learning_-rate_-decay \times \frac{global_-step}{learning_-rate_-batch_-size} $$
其中，$learning_-rate_-base$为学习率初始值，$learning_-rate_-decay$为学习率衰减率,$global_-step$记录了当前训练轮数，为不可训练型参数。学习率$learning_-rate$更新频率为输入数据集总样本数除以每次喂入样本数。
代码表示如下：

'''指数衰减学习率'''
learning_rate_base =0.2 #为学习率初始值
#设置学学习率衰减率,设置为略小于1的数，如0.99,0.98，不能超过1。
learning_rate_decay = 0.99 
learning_rate_step = 1 #更新学习率的频率
#记录了当前训练轮数，初值设为0，设置为不被训练
global_step =tf.Variable(0,trainable= False)
#定义指数衰减学习率，学习率公式见上式,
#若staircase 设置为 True 时，表示 global_step/learning_rate_step取整数，学习率阶梯型衰减；
#若staircase设置为false时，学习率会是一条平滑下降的曲线。
learning_rate = tf.train.exponential_decay(learning_rate_base,global_step,learning_rate_step,learning_rate_decay,staircase=True)
#定义待优化参数，初始值为2
w = tf.Variable(tf.constant(2,dtype=tf.float32 ))
#定义损失函数loss = （w+1）^2
loss = tf.square( w + 1)
#定义反向传播方法
train_step=tf.train.GradientDescentOptimizer(learning_rate).minimize(loss,global_step=global_step)

with tf.Session() as sess:
    #初始化参数
    init_op = tf.global_variables_initializer()
    sess.run(init_op)
    #迭代20次
    STEPS = 20
    for i in range(STEPS):
        sess.run(train_step)
        learning_rate_value = sess.run(learning_rate)
        global_step_value = sess.run(global_step)
        w_value = sess.run(w)
        loss_value = sess.run(loss)
        print("迭代{}次，global_step为{}，学习率为{}，w的值为：{}f，损失值为：{}f".format(i,global_step_value,learning_rate_value,w_value,loss_value))

5.实战反向传播

反向传播介绍的差不多了，我们来个小例子深入理解一下反向传播。

5.1例子1

求解$ y=2*x +5.0$中的权重（weight）2 和偏置（biases）5。

5.1.1导入模块，生成数据集

导入模块，生成数据集，生成标准数据集。

import tensorflow as tf
import numpy as np
learning_rate = 0.01  #定义学习率
momentum = 0.1#定义momentum 优化器中的参数

x = np.random.rand(100).astype(np.float32)#生成数据集
y_ = 2*x +5.0#生成标准值，用于与预测值对比

weight =tf.Variable(tf.zeros([1]))#定义权重（weight），初始值随意，这里定义为0
biases = tf.Variable(tf.zeros([1]))#定义偏置（bias），初始值随意，这里定义为0

5.1.2前向传播过程

如果是神经网络过程，就需要定义输入，输出，隐藏层。而我们现在这个例子，就相当于1个神经元，$x$为输入，$y$为输出，所以直接生成$y$即为前向传播过程。

y = weight*x +biases#生成预测值，即前向传播过程

5.1.3反向传播过程

定义损失函数、生成损失值（loss)、选择优化器优化损失值。
定义损失函数:

'''定义损失函数'''
loss_mse = tf.reduce_mean(tf.square( y - y_))#用均方误差（MSE)计算损失值

优化器有3种选择，我们现在都写出来，选择其中的1种来优化就好：

1.使用梯度下降方法优化loss值:

#使用梯度下降方法优化loss值
train_step=tf.train.GradientDescentOptimizer(learning_rate).minimize(loss_mse)

2.使用momentum 优化器方法优化loss值:

#momentum 优化器
train_step=tf.train.MomentumOptimizer(learning_rate,momentum).minimize(loss_mse)

3.使用adam 优化器方法优化loss值:

#adam 优化器
train_step=tf.train.AdamOptimizer(learning_rate).minimize(loss_mse)

5.1.4生成会话，训练 STEPS 轮

初始化变量，训练，生成优化后的权重$（weight）$和偏置$（biases）$。每100次打印一次权重$（weight）$和偏置$（biases）$。

with tf.Session() as sess:
    #初始化变量
    init_op = tf.global_variables_initializer()
    sess.run(init_op)
    
    #训练 STEPS 轮
    STEPS =10000
    for i in range(STEPS):
        sess.run(train_step)
        #每100次打印一次权重（weight）和偏置（biases）。
        if i % 500 == 0:
            print(i,sess.run(weight),sess.run(biases))
